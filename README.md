## 專案名稱: ME (專案開始時間點: 2026/05月)

```
「ME」不是單純的一個專案，而是一份送給家人的禮物。
我想在 VR 世界裡打造一個屬於「我」的 NPC，讓這個由我蒸餾出來的分身，未來能夠陪家人在虛擬世界裡互動、對話，甚至一起遊玩我親手建構的世界

這個世界的起點
目前我正在一步步打造這個計畫的底座。
像是 Line 機器人，它就像是我日常資料蒐集與自我紀錄的前置作業，幫助我把生活裡的片段慢慢整理成可被理解、可被蒸餾的素材。
而 VR 世界 Unity 則是這個夢想真正展開的舞台，讓「我」能在虛擬空間裡擁有自己的存在感。

未來的樣子
未來還會有一個「蒸餾版的我」手機 App。
這不只是聊天工具，而是承接前期資料整理、逐步形成我個人語氣與思考方式的載體；等到 AI 開始說話時，也會盡可能使用接近我本人的聲線，讓家人聽見的不只是回答，而像是我真的回來了。
```
